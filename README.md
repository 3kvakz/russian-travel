# Путешествие по России

Страница, где мы выбрали и описали пять интересных мест в России, достойные вашего отпуска:
* Куршская коса
* Кольский полуостров
* Алтай
* Зимний Байкал
* Карелия

Написана на HTML (52%) и CSS (48%) с четырьмя интервалами:
* 1280—∞
* 1024—1279
* 768—1023
* 0—767
